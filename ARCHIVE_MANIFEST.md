# Archive Manifest

This archive contains an exploratory PocketFlow Creator project package.

## Included

- Complete docs directory with design, requirements, wireframes, architecture, and implementation plan.
- Starter PySide6 GUI shell.
- Data model, validator, generator, provider interface stubs.
- Example document summarizer project.
- Cross-platform setup, run, test, lint, and format scripts.
- Unit tests for core non-GUI behavior.
- Architecture SVG diagram.

## Not Yet Implemented

- Full QGraphicsView graph designer.
- Real drag/drop node placement.
- Real property grid editor.
- Real prompt/YAML/Python syntax highlighting.
- Real Ollama HTTP integration.
- Real PocketFlow execution/debug runner.
- Full project YAML loader/saver.

These are intentionally left as implementation phases rather than incomplete hidden behavior.
