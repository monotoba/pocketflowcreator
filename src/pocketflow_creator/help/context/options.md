# Options

**Tools > Options** controls appearance and language settings.

## Appearance

Choose how the application determines its color scheme:

| Mode | Behavior |
|---|---|
| **System (follow OS setting)** | Uses your operating system's dark/light preference. Changes automatically when you change your OS theme. |
| **Light** | Always uses the light theme regardless of OS settings. |
| **Dark** | Always uses the dark theme regardless of OS settings. |

The theme change takes effect immediately after clicking **OK**.

## Language

Choose the display language for all menus, dialogs, and labels.

| Locale | Language |
|---|---|
| System default | Follows your operating system locale |
| English | English |
| Español | Spanish |
| Français | French |
| Deutsch | German |
| 中文 | Chinese (Simplified) |
| 日本語 | Japanese |

**A restart is required** after changing the language. Click OK, then restart the application.

## Help content

Help pages and tutorial content are always displayed in English regardless of the
selected UI language.
